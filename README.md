This project presents a comprehensive Uber Trip Analysis Dashboard built in Power BI to analyze ride-booking patterns, revenue generation, trip efficiency, customer behavior, vehicle utilization, and location-based insights.

The solution was developed to transform raw trip data into actionable business intelligence, enabling stakeholders to make data-driven decisions related to demand forecasting, pricing optimization, fleet allocation, and operational efficiency.

The dashboard consists of three interactive report pages:

**1. Overview Analysis**

**2. Time Analysis**

**3. Detailed Drill-Through Analysis**

**Key Performance Indicators (KPIs) -**

Total Bookings -	Total trips completed
Total Booking Value -	Total revenue generated
Average Booking Value	- Revenue per booking
Total Trip Distance -	Total distance travelled
Average Trip Distance	- Average distance per trip
Average Trip Time -	Average trip duration

**Power BI Techniques Used -**

- Data Modeling
- Star-schema design
- Relationship management
- Inactive relationship handling
- DAX Measures
- KPI calculations
- Dynamic measure switching
- Conditional calculations
- Dynamic titles
- Visualizations
- KPI Cards
- Donut Charts
- Area Charts
- Line Charts
- Matrix Heatmaps
- Bar Charts
- Tables
- Drill-through Reports
